What is Iteration?
Loops let you repeat a block of code multiple times ‚Äî the heart of efficient programming
Definition

Iterative Process

Repeating a set of instructions a number of times until a condition is met. Instead of writing the same code 100 times, you write it once inside a loop!

Why Use Loops?

Power of Repetition

Print 1 to 1000, sum 50 numbers, check every character in a string ‚Äî all impossible without loops. They make programs short and powerful.

Types in Java

3 Loop Types

Java has for, while, and do-while loops. Each works differently but all repeat code based on a condition.

‚öôÔ∏è 4 Essential Parts of Every Loop

1. Initialisation

Set the starting value of the loop variable. Runs only once.
int i = 1;

2. Test Condition

Checked before each iteration. Loop continues while true, stops when false.
i <= 10

3. Body

The code that actually runs in each iteration. Can be one or many statements.

4. Update

Increments or decrements the variable. Without this ‚Üí infinite loop!
i++

Entry-Controlled Loops

Condition Checked First

The condition is tested before the loop body runs. If false from the start, the body never executes. for and while are entry-controlled.

Exit-Controlled Loops

Runs At Least Once

The condition is tested after the body runs. Even if the condition is false, the loop body executes at least once. do-while is exit-controlled.

Infinite Loop

Never-Ending Loop

A loop where the condition never becomes false. Usually a bug! Example: forgetting to update the variable, or writing while(true).

The for Loop
Best used when the number of iterations is known in advance

The while Loop
Best when iterations depend on a condition, not a fixed count
Syntax

Structure

// Initialise BEFORE the loop
int i = 1;

while (condition) {
    // body
    // Update INSIDE the loop
    i++;
}
Key Point

Entry Controlled

The condition is checked before entering the loop. If the condition is false from the start, the loop body never runs ‚Äî not even once.

int x = 100;
while (x < 10) {  // false!
    System.out.println(x);
}
// Nothing prints
Example 1

Print 1 to 5

int i = 1;          // init
while (i <= 5) {   // condition
    System.out.println(i);
    i++;            // update
}
// Output: 1 2 3 4 5
Example 2

Factorial of n

int n = 5, fact = 1, i = 1;
while (i <= n) {
    fact = fact * i;
    i++;
}
System.out.println("5! = " + fact);
// Output: 5! = 120
Example 3

Digits of a Number

int n = 1234, count = 0;
while (n != 0) {
    n = n / 10;
    count++;
}
System.out.println("Digits: " + count);
// Output: Digits: 4

The do-while Loop
Guarantees the body runs at least once ‚Äî condition checked at the end
Syntax

Structure

int i = 1;
do {
    // body ‚Äî runs FIRST
    i++;
} while (condition); // ‚Üê semicolon!

‚ö†Ô∏è Don't forget the semicolon ; after while(...)!

Key Difference

Always Runs Once

Even if the condition is false, the body executes once. This makes do-while perfect for menu-driven programs where you always want to show the menu before checking input.

int x = 100;
do {
    System.out.println(x); // prints!
} while (x < 10); // false, but too late
// Output: 100
Example 1

Print 1 to 5

int i = 1;
do {
    System.out.println(i);
    i++;
} while (i <= 5);
// Output: 1 2 3 4 5
Example 2

Sum until input = 0

int sum = 0, n;
do {
    // accept input n from user
    sum += n;
} while (n != 0);
System.out.println("Sum = " + sum);
Vs While

The Critical Difference

while ‚Üí condition checked BEFORE body
do-while ‚Üí body runs THEN condition checked
// while: 0 iterations possible
// do-while: minimum 1 iteration

Jump Statements
break and continue ‚Äî control the flow inside loops
break

Exits the Loop

Immediately terminates the loop and moves to the statement after the loop. Like an emergency exit.

for (int i = 1; i <= 10; i++) {
    if (i == 5) break;
    System.out.print(i + " ");
}
// Output: 1 2 3 4
continue

Skips Current Iteration

Skips the rest of the current iteration and jumps to the next one. The loop doesn't stop ‚Äî it continues.

for (int i = 1; i <= 10; i++) {
    if (i % 2 == 0) continue;
    System.out.print(i + " ");
}
// Output: 1 3 5 7 9
